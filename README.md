# Click-Cart - E-Commerce Website

Overview

Click Cart is a full-stack e-commerce web application that allows users to browse products, add items to the cart, manage orders, and securely authenticate their accounts. The project demonstrates modern web development practices and showcases frontend, backend, database, and authentication skills.

Features --

User Features -

User Registration and Login
Secure Authentication
Browse Products
Search Products
Product Categories
Product Details Page
Add to Cart
Update Cart Quantity
Remove Items from Cart
Checkout Process
Order History
Responsive Design

Admin Features -

Add New Products
Update Product Information
Delete Products
Manage Orders
View Customer Details
Dashboard for Product Management

Tech Stack --

Frontend -

HTML
CSS
JavaScript
React.js


Backend -

Node.js
Express.js


Database -

MongoDB


Authentication -

JWT (JSON Web Token)
bcrypt Password Hashing
